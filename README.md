# Weather Dashboard

## Purpose
A website that queries Open Weather api and geo location to display weather information for a given city and forecast, as well as stores previous searches locally.

## Built With
* HTML
* CSS
    - Bootstrap
* JavaScript
    - jQuery
    - Moment.js
* OpenWeather APIs
    - One Call API
    - Geocoding API
